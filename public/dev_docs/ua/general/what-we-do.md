# Що ми робимо

- **Відкрита платформа** для спільнот, щоб вони могли запускати власні мережі.
- **Сучасний UX** з тематикою та ресурсами на рівні домену.
- **Інструменти для творців**: підписки, фонди для постів (нагороди у стилі гри), NFT, промоакції.
- **Управління**: конфігурація на ланцюгу та голосування стейкерів.
- **Інтероперабельність**: мережі використовують однакову токеноміку, але зберігають свою культуру.
