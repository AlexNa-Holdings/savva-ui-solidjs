# Руководство по установке - Обзор

Это руководство проведёт вас через процесс настройки собственной сети SAVVA с нуля. Следуя этим инструкциям, вы сможете запустить полноценный экземпляр SAVVA с серверной частью и пользовательским интерфейсом.

## Что вы развернёте

Полная сеть SAVVA состоит из:

1. **Backend Server** - сервер API на Go, который отвечает за:
   - Аутентификацию пользователей и сессии
   - Сохранение и получение постов
   - Интеграцию с IPFS
   - Управление базой данных
   - WebSocket-соединения
   - Взаимодействие с блокчейном

2. **UI Website** - фронтенд на SolidJS, который предоставляет:
   - Интерфейс для создания и просмотра контента
   - Интеграцию с Web3-кошельками
   - Загрузку файлов в IPFS
   - Взаимодействие с умными контрактами
   - Многоязычную поддержку

3. **Smart Contracts** :
   - Контракт токена SAVVA
   - Контракт стекинга
   - Контракт управления
   - Контракт NFT для контента
   - И другие...

## Installation Flow

## Requirements Overview

- **Server**: Linux-сервер (рекомендуется Ubuntu 20.04+)
- **Database**: PostgreSQL 14+ (или управляемый сервис базы данных)
- **IPFS**: Локальный узел IPFS + внешняя служба пиннинга с публичным шлюзом
- **Web Server**: Nginx или Apache
- **Domain**: Имя домена с SSL-сертификатом
- **Blockchain**: Доступ по RPC к совместимой с Ethereum сети (рекомендуется WSS)
