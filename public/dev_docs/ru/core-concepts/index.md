# Основные концепции

В этом разделе описывается основная архитектура приложения SAVVA на SolidJS: как оно подключается к бэкенду, как работают домены и ресурсы, как интерфейс управляется конфигурацией для каждого домена и многое другое.

## Темы

- [App Initialization](/docs/core-concepts/app-init) - Как приложение запускается и находит правильный бэкенд и домен
- [Authentication](/docs/core-concepts/auth) - Аутентификация пользователя и подключение кошелька
- [Configuration Contract](/docs/core-concepts/config-contract) - Как `/info` определяет поведение во время выполнения
- [Connection Orchestration](/docs/core-concepts/connection-orchestration) - Управление соединениями WebSocket и HTTP
- [Domains](/docs/core-concepts/domain) - Конфигурация доменов и разрешение ресурсов
- [Publishing Posts](/docs/core-concepts/publishing-posts) - Как создаются и публикуются посты в IPFS
- [Showing Posts](/docs/core-concepts/showing-posts) - Как посты отображаются и рендерятся
- [Encrypted Posts](/docs/core-concepts/encrypted-posts) - Сквозное шифрование для контента, доступного только подписчикам
- [User Profile](/docs/core-concepts/user-profile) - Управление профилем пользователя и данные в блокчейне
