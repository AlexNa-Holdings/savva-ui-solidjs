# Conceptos principales

Esta sección cubre la arquitectura central de la aplicación SAVVA SolidJS: cómo se conecta al backend, cómo funcionan los dominios y los activos, cómo la interfaz de usuario está impulsada por la configuración por dominio, y más.

Lo que cubriremos:

- Cómo arranca la aplicación y encuentra el backend y el dominio correctos
- Cómo `/info` dirige el comportamiento en tiempo de ejecución (dominios, ID de cadena, pasarelas IPFS, activos)
- Cómo se resuelven los activos de dominio y cuándo se utiliza el paquete predeterminado
