- [Contrato de configuración](/docs/core-concepts/config-contract) - Cómo `/info` determina el comportamiento en tiempo de ejecución
- [Orquestación de conexiones](/docs/core-concepts/connection-orchestration) - Gestión de conexiones WebSocket y HTTP
- [Dominios](/docs/core-concepts/domain) - Configuración de dominios y resolución de activos
- [Publicación de entradas](/docs/core-concepts/publishing-posts) - Cómo se crean las entradas y se publican en IPFS
- [Mostrar publicaciones](/docs/core-concepts/showing-posts) - Cómo se muestran y renderizan las publicaciones
- [Publicaciones cifradas](/docs/core-concepts/encrypted-posts) - Cifrado de extremo a extremo para contenido exclusivo para suscriptores
- [Perfil de usuario](/docs/core-concepts/user-profile) - Gestión del perfil de usuario y datos on-chain
